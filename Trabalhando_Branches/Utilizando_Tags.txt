## Uma tag e um marco no desenvolvimento, para criar utilizamos o comando git tag -a <nome> -m <menssagem>

## Para mostrar as tags utlizamos o comando git tag

## Para mostrar o conteudo de uma tag use git show <nome>

## Para passear entre as tag utilizando o git checkout <nome>

## Para enviar uma tag especifica para o repositorio use git push origin <tag>
