##  O comando git stash faz com que todas as suas alteracoes  desde o ultimo commit sejam  desfeitas  e uma versao desse codigo tenha uma versao salva
 
 # git stash 


 ## Para vermos as stashs criadas  utilizamos o comando git stash list 
 

 ## Para que vc consiga aplicar a stash vc usa o comando git stash apply +   o numero da stash

 ## Para mostrar as alteracoes da stash use o comando git stash show -p +  numero da stash


 ## Para remover uma stash utilize git stash clear ou para remover uma so utilize o git stash drop + nome
